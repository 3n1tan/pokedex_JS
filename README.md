# Project title

PokeDex
The Pokedex Application is a web-based tool designed using HTML, CSS, and JavaScript, with real-time data fetched from an online API. This user-friendly application serves as a digital encyclopedia for Pok√©mon enthusiasts, allowing users to search for various species of pokemon by ID, name, types and generation

## Setup and usage

- At the web app front page, select from the drop down select tag to choose which geenration of 
pokemon species that needs to be displayed.

- Once the generation displayed, one can filter the results by name, types.
